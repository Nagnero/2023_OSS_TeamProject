# Progress Report for Team #10

## Progress Summary

주식투자를 할 때 필요한 자료들이 정말 많다. 그러나 국내에는 이러한 정보들을 한 번에 제공하는 사이트가 없다. 그래서 web crawling을 통해 여러 사이트로부터 이러한 정보들을 긁어와 web 또는 excel을 통해 한 번에 제공하는 사이트를 개발할 계획이다. <br>

- 실시간 종목 조회 순위를 기반으로 해당 종목의 이슈를 제공
- 투자를 할 때 필요한 여러가지 지표를 한 번에 모아서 제공 ex) 공포지수, 나스닥지수, 원자재 가격 등등
